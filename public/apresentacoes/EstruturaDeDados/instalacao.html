Instalação

1 Clone o Repositório

Baixe o código do GitHub para sua máquina local.

git clone https://github.com/seu-usuario/Elevador.git cd Elevador

2 Configure o Ambiente

Instale o JDK 8 ou superior e o Maven.

# Instale o JDK sudo apt install openjdk-11-jdk # Instale o Maven sudo apt install maven

3 Compile e Execute

Compile o projeto e inicie a aplicação.

mvn clean install mvn exec:java -Dexec.mainClass="Main"

4 Acesse a Aplicação

A interface gráfica será iniciada automaticamente após a execução do comando acima. A aplicação permitirá configurar os parâmetros da simulação e visualizar os elevadores em funcionamento.

Instalação

1

Clone o Repositório

Baixe o código do GitHub para sua máquina local

git clone https://github.com/seu-usuario/confloto.git
cd confloto
2

Configure o Backend

Instale as dependências e configure o Firebase

cd backend
npm install
# Configure seu arquivo .env com as credenciais do Firebase
npm run dev
3

Configure o Frontend

Instale as dependências e inicie o servidor de desenvolvimento

cd frontend
npm install
# Configure seu arquivo .env com a URL do backend
npm run dev
4

Acesse a Aplicação

Abra a aplicação no seu navegador

  • Backend: http://localhost:5000
  • Frontend: http://localhost:8000
